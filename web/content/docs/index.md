# Overview

This documentation provides information on the features and design for the SpecAtlas.

The SpecAtlas is an application that provides a catalog of technical specifications and their relationships. The goal
is to provide a comprehensive reference point for discovering and learning about specifications. It is intended to be
a living and evolving reference, with a minimal delay between publication of a standard and inclusion in the Atlas.
This evolution of the SpecAtlas is focused specifically on specifications related to all aspects of Digital Identity,
however the design is intended to be generalizable to any family of specifications.

The SpecAtlas currently includes specs such as:

- IETF RFCs
- ISO, IEC, and ISO/IEC International Standards
- ITU-T Recommendations
- OpenID Foundation Specifications
- W3C Recommendations
- OASIS Standards

> [!NOTE]
> Given the diversity of organizations developing specifications, terminology used in the SpecAtlas may differ from those
> used by individual standards organizations. This documentation provides specific meanings for terminology within the database.
