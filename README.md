# SpecAtlas

The SpecAtlas is a reference source for technical specifications, protocols, and associated technologies. The core of the
Atlas is a graph database, with a Web presence to consume the data.

The current evolution of the SpecAtlas is focused on Digital Identity technologies, but the framework is extensible to
support specifications across the digital estate.

Specifications profiled by the Atlas include those from:

- ISO, IEC, and ISO/IEC
- ITU-T
- IETF
- OpenID Foundation
